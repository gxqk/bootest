## âš™ï¸ Building the Bootstrapper

### Visual Studio 2022 Method (Recommended)

1. Open the solution in Visual Studio 2022
2. Right-click on the project
3. Click "Publish"
4. Create new profile:
   - Target: Folder
   - Configuration: Release
   - Target Framework: net7.0-windows
   - Deployment Mode: Self-contained
   - Target Runtime: win-x64
   - File Format: Single file
5. Click "Publish"

Your standalone .exe will be created at:
```
src\bin\Release\net7.0-windows\win-x64\publish\bootstrapper.exe
```

ğŸ’¡ Tip: This is a single file that contains everything needed to run on any Windows 64-bit PC!

### Project Configuration

```xml
<PropertyGroup>
  <OutputType>WinExe</OutputType>
  <TargetFramework>net7.0-windows</TargetFramework>
  <PublishSingleFile>true</PublishSingleFile>
  <SelfContained>true</SelfContained>
  <RuntimeIdentifier>win-x64</RuntimeIdentifier>
  <EnableWindowsTargeting>true</EnableWindowsTargeting>
  <IncludeNativeLibrariesForSelfExtract>true</IncludeNativeLibrariesForSelfExtract>
</PropertyGroup>
```
